# 數值模型檢核工具 - 快速參考

## 🚀 快速開始

1. 啟動本地服務器：`python -m http.server 8000`
2. 訪問：`http://localhost:8000/start.html`
3. 點擊「4. 數值模型檢核」
4. 選擇需要的分析功能

---

## 📊 四大核心功能

### A. 時期進度模擬
**一鍵操作**：點擊「開始模擬」

**關鍵數據**：
- 修煉至金仙：7880 年（5.47 天）
- 平均每時期：656.7 年（10.9 小時）
- 第1時期：80 年（1.33 小時）
- 第12時期：1200 年（20 小時）

---

### B. 資源上限計算器
**輸入參數**：
- 道心值（0-999+）
- 道證值（0-999+）
- 目標時期（1-12）

**計算公式**：
```
道心加成 = 道心 × 0.1%
道證加成 = 道證 × 1%
最終上限 = 基礎 × (1 + 道心加成) × (1 + 道證加成)
```

**示例**：
```
道心100 + 道證20 = 32%總加成
靈力：100 → 132
金錢：1000 → 1320
```

---

### C. 建築投資回報分析
**一鍵操作**：點擊「分析建築 ROI」

**ROI 評級**：
- 🟢 優秀（≥1.0）：優先建造
- 🟡 中等（0.5-1.0）：適度推薦
- 🔴 較低（<0.5）：後期考慮

**建議順序**：
1. 高 ROI + 低成本
2. 高權重（後期收益大）
3. 上限類（解決瓶頸）

---

### D. 功法效益分析
**一鍵操作**：點擊「分析功法效益」

**推薦度**：
- 🌟 強烈推薦（≥20）：優先學習
- ⭐ 適度推薦（5-20）：按需學習
- ⚠️ 謹慎考慮（<5）：非必要不學

**功法類型**：
- **上限類**：立即見效，解決溢出
- **倍率類**：後期收益，長期投資
- **產率類**：持續收益，早期發展

## 🎯 常見問題

**Q: 遊戲進度太慢？**
A: 檢查建築 ROI，優先建造高產出建築

**Q: 資源經常溢出？**
A: 建造上限建築或學習上限功法

**Q: 不知道學什麼功法？**
A: 查看功法效益分析，選擇高性價比功法

**Q: 何時輪迴最划算？**
A: 計算當前建築總數 ÷ 10 = 道心收益

## 📈 關鍵數值

### 壽元公式
```
第1時期：80 年
第N時期：前一時期 + N × 100 年
```

### 道心道證
```
道心：建築總數 ÷ 10
道證：建築總數 ÷ 50
道心加成：0.1% / 點
道證加成：1% / 點
```

### 建築產出
```
產率 = 基礎 × (等級 + 權重)²
成本 = 基礎 × 成本係數^等級
```

### 功法性價比
```
上限類：效果值 ÷ 技能點
倍率類：(倍率-1) × 100 ÷ 技能點
產率類：產率增加 ÷ 技能點
```
